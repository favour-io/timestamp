⏱️ Timestamp Microservice

This is a simple timestamp API compatible with the freeCodeCamp project.

Endpoints

  • /api — Current time
  • /api/<date> — A date string or Unix timestamp

Examples

GET /api                     → {"unix": 1697040000000, "utc": "Mon, 11 Oct 2023 00:00:00 GMT"}
GET /api/2015-12-25          → {"unix": 1451001600000, "utc": "Fri, 25 Dec 2015 00:00:00 GMT"}
GET /api/1451001600000       → {"unix": 1451001600000, "utc": "Fri, 25 Dec 2015 00:00:00 GMT"}
GET /api/invalid-date        → {"error": "Invalid Date"}
